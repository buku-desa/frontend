# Warga FE Baru

Proyek starter TypeScript + React (.tsx) menggunakan Vite.

Cara pakai singkat:

1. Install dependencies:

   npm install

2. Jalankan dev server:

   npm run dev

3. Build produksi:

   npm run build

4. Preview build:

   npm run preview


Catatan: desain minimal. Ubah konfigurasi `tsconfig.json` dan ESLint sesuai kebutuhan.
